# LeForge SDKs

Client libraries for interacting with LeForge services.

## Available SDKs

- [JavaScript/TypeScript](./javascript) - For Node.js and browsers
- [Python](./python) - For Python applications
- [.NET](./dotnet) - For C# and .NET applications

## Installation

### JavaScript

```bash
npm install @LeForge/sdk
```

### Python

```bash
pip install LeForge-sdk
```

### .NET

```bash
dotnet add package LeForge.SDK
```
